python的new和init

发表于 2019-01-07 | 分类于 python ， 特殊方法 | 阅读次数:

new和init一样都是特殊方法，new的调用发生的更早的一点，new方法应该返回对象的实例，之后会调用init方法，此方法不应该有返回值或者返回值为None
如果new方法没有返回实例，init方法不会触发。new方法通常被用在自定义的类中，比如自定义的类继承自不可变对象（int，string， tuple），比如一个自定义的int类型只能为正。还有一方面python官方提到在元编程中很有用，不过目前不会元编程，下次学时的时候再补上吧

容斥原理

发表于 2018-12-27 | 阅读次数:

容斥原理主要用来搭配逆向思维来解决问题
公式很简单: 两个集合是 $|A \bigcup B| = |A| + |B| - |A \bigcap B|$
三个集合是 $|A \bigcup B \bigcup C| = |A|+|B|+|C|-|A \bigcap B|-|B \bigcap C|-|A \bigcap C|+|A \bigcap B \bigcap C|$
$|\bigcap_{n}^{i=1} A_{i}|=\sum_{k=1}^{n}(-1)^{k+1}(\sum_{1\leqslant i_1 < … <i_k \leqslant n} |A_{i_1} \bigcap … \bigcap A_{i_k}|)$

积水问题

发表于 2018-12-20 | 阅读次数:

昨天面试一道积水问题的题目,以为做对了,回去仔细想了一下还是有问题.其实静下心来想很清楚,只用考虑每个柱子上到底能不能积水.
第一种直接思考,每根柱子如果小于左边的最大值并且大于右边的最小值,那它上面肯定会有积水,所以两个数组保存左右方向的最大值就行,时间复杂度为O(n),空间复杂度也为O(n)
第二种也是第一种的变形,只是代码稍微不一样,遍历一次就行,空间复杂度为O(1),时间复杂度为O(n)
